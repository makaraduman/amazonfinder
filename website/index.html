Amazon Product Finder

Definition

"An application is a dedicated non-browser client for AWS. Operationally, an application is structured like and behaves similarly to a dynamic site. Again, caching should be used to increase responsiveness and availability."

Introduction

Amazon product finder is targeted mainly for the home user. Application should give its user a fast way to search the Amazon catalog for products with given search parameters. Amazon.com has enabled third party developers to develop applications that are using the ever innovating Amazon Web Services API. This little application is a mock-up version that hopefully will evolve into something useful. The UI is ready and miscellaneous functions, procedures are in their place.. But there's a lot more work to be done.
